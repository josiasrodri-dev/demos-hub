# Developer Demos Hub

A lightweight static landing page created to organize and provide access to live demonstrations of my open-source web projects.

This repository works as a central entry point for projects that require PHP hosting, while my main portfolio is deployed separately.

## Purpose

Since PHP-based projects cannot be deployed directly through my portfolio hosting environment, this hub provides a simple and accessible way to showcase their live demos through clean URLs.

Each project is hosted independently and linked from this page.

## Features

- Clean project navigation
- Responsive design
- Organized demo links
- Lightweight static implementation
- Custom project URLs

## Tech Stack

- HTML5
- CSS3

## Project Structure

```text
assets/
    icons/
    images/
project-name/
    project files
icons.css
style.css
index.html
LICENSE
README.md
```

## Deployment

This project is deployed as a static website and serves as the public entry point for PHP-based project demonstrations.

## Related Projects

The projects included in this hub are available as individual repositories and demonstrate different web development skills, including backend development with PHP and database integration with MySQL.
